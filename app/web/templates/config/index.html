Tailscale Funnel Setup Guide

Tailscale Funnel exposes this dashboard to the internet so remote agents can POST to it — no VPN, no domain, free forever. Install it once on this server; agents are just HTTP clients.

  1. Create a free account at tailscale.com (no credit card required). Choose the Personal plan.
  2. SSH into this server and install Tailscale: 
    curl -fsSL https://tailscale.com/install.sh | sh
  3. Log this server into your tailnet: 
    sudo tailscale up
    Copy the printed URL, open it in any browser, and click Connect.
  4. Enable HTTPS in the Tailscale admin console: Go to login.tailscale.com/admin/dns → HTTPS Certificates → Enable HTTPS.
  5. Allow Funnel in your ACL policy: Go to login.tailscale.com/admin/acls and add this block inside the top-level object, then click Save: 
    "nodeAttrs": [
    {"target": ["*"], "attr": ["funnel"]},
],
  6. Expose the dashboard port (survives reboots): 
    sudo tailscale funnel --bg 7070
    Tailscale will print your permanent public URL, e.g. https://yourvm.taild0c836.ts.net.
  7. Paste that URL into the Public Dashboard URL field above and save.

Verify after reboot: run tailscale funnel status — the funnel should still be active. If not, re-run sudo tailscale funnel --bg 7070 once more.
